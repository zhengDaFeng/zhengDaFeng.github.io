# 写在前面

很早之前就想做个属于自己的导航了，原因有很多。

1. 不同浏览器的收藏不互通，导致想体验不同浏览器的成本较高。
2. 浏览器上的收藏栏太小了，容纳不了日益提升的网址收藏数量。
3. 装逼。

# 历程

一开始是用的 VPS 做服务器弄的网站，因为是国外的服务器，速度...为什么是国外的，嘿嘿嘿...

然后再搭建博客的时候，发现可以使用 GitHub Pages 搭建博客，所以，为什么不用它搭建个导航网站呢。至于道德问题，个人不盈利网站，口下留情。
